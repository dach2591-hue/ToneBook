## First Steps

### 1. Create Your First Song
- Click "+ Create Song"
- Add title, artist, key
- Add chords with `[C]`, `[Am]`, etc.
- Use `[Verse 1]`, `[Chorus]` for sections
- Add `-- comments` for performance notes
- Click "Save Song"

### 2. Organize
- Songs automatically grouped by artist
- Create custom folders for setlists
- Pin songs to create ordered setlists

### 3. Search
- Type in search box to find any song
- Searches all songs by title, artist, or key

### 4. Performance Mode
- Click "Performance" button
- Clean view for live performance
- Transpose chords in real-time
- Adjust font size

### 5. Export Backup
- Click "Import/Export"
- Click "Export"
- Choose "Export All Songs"
- Save the .tone file somewhere safe

```

```


### Comments
```

```
